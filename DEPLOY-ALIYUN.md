# 在阿里云上部署 Pomor 官网（下一步怎么做）

你已经在阿里云买了网站/主机，按下面顺序做即可。

---

## 第一步：确认你买的是哪种产品

登录 [阿里云控制台](https://home.console.aliyun.com/)，看左侧菜单里有没有这些：

| 产品 | 常见入口 | 上传方式 |
|------|----------|----------|
| **云虚拟主机** | 产品与服务 → 云虚拟主机 | 控制台里的「文件管理」或 FTP |
| **轻量应用服务器** | 产品与服务 → 轻量应用服务器 | 宝塔面板 / FTP / 远程桌面传文件 |
| **ECS 云服务器** | 产品与服务 → 云服务器 ECS | 自己装 Nginx，用 FTP/SFTP 传文件 |

记下你的**网站根目录**是哪个路径（虚拟主机一般是 `htdocs` 或 `wwwroot`；轻量/ECS 若装了宝塔，常见是 `/www/wwwroot/你的域名/`）。

---

## 第二步：在本机先准备好要上传的内容

1. **官网页面**  
   用项目里的 **`web/`** 目录，里面要有：
   - `index.html`
   - `style.css`
   - `app.js`
   - `assets/` 文件夹（若有）
   - 其他 `web/` 下所有文件

2. **（可选）安装包**  
   在项目根目录执行 `npm run dist`，在 **`release/`** 里会生成 **`Pomor Timer Setup 1.0.0.exe`**（版本号以你 `package.json` 为准）。  
   若你打算用「你的域名/download/xxx.exe」这种地址，就先记下：**你的域名** 和 **exe 文件名**。

3. **配置下载链接**  
   打开 **`web/app.js`**，找到 **`DOWNLOAD_URL`**（约第 89 行），改成你打算放 exe 的地址，例如：
   ```js
   const DOWNLOAD_URL = 'https://你的域名/download/Pomor-Timer-Setup-1.0.0.exe';
   ```
   若暂时不提供下载，可先留空 `''`，页面会显示「即将上线」；以后放了 exe 再改这里并重新上传 `app.js`。  
   改完后保存。

---

## 第三步：把官网文件上传到「网站根目录」

- **云虚拟主机**：在控制台进入「文件管理」→ 进入网站根目录（如 `htdocs` 或 `wwwroot`）→ 上传 **`web/`** 里的**所有文件**（不要把整个 `web` 文件夹传上去，而是把 `index.html`、`style.css`、`app.js`、`assets` 等放在根目录下）。
- **轻量应用服务器 / ECS**：  
  - 若装了**宝塔**：网站 → 你的站点 → 根目录 → 上传同样这些文件到根目录。  
  - 若用 **FTP/SFTP**：用 FileZilla 等工具连上主机，把 `web/` 里的文件上传到网站根目录。

上传完成后，目录里应能看到 `index.html`、`style.css`、`app.js` 等，且 `index.html` 在根目录（不在子文件夹里）。

---

## 第四步：（可选）放安装包，让别人能下载

若你要提供 exe 下载：

1. 在网站根目录下新建一个文件夹，例如 **`download`**。
2. 把本机 **`release/`** 里的 **`Pomor Timer Setup 1.0.0.exe`** 上传到这个 `download` 文件夹。
3. 确认 **`web/app.js`** 里的 **`DOWNLOAD_URL`** 是这个地址（第三步里已改过的话就不用再改）：
   ```text
   https://你的域名/download/Pomor-Timer-Setup-1.0.0.exe
   ```
   若你改过 `DOWNLOAD_URL` 但还没上传过 `app.js`，上传一次最新的 `app.js` 到网站根目录覆盖旧文件。

---

## 第五步：用浏览器检查

1. 打开浏览器，访问 **你的域名**（如 `https://你的域名.com`）。  
   应能看到 Pomor 官网页面（大标题、介绍、下载按钮等）。
2. 若配置了下载链接，点击「下载 Windows 安装包」应能下载 exe；若暂未配置，会显示「即将上线」或提示未配置，属正常。

若打不开页面，可检查：域名是否已解析到这台主机、主机是否绑定了该域名、文件是否真的在「网站根目录」（控制台/宝塔里写的那个路径）。

---

## 小结（你要做的几件事）

| 顺序 | 做什么 |
|------|--------|
| 1 | 在阿里云控制台确认是「虚拟主机 / 轻量 / ECS」，并找到「网站根目录」和上传方式 |
| 2 | 本机改好 `web/app.js` 里的 `DOWNLOAD_URL`（或先留空），保存 |
| 3 | 把 `web/` 下的所有文件上传到网站根目录，保证根目录下有 `index.html` |
| 4 | （可选）在根目录下建 `download`，上传 exe，并确认 `DOWNLOAD_URL` 和实际路径一致 |
| 5 | 浏览器访问你的域名，检查页面和下载是否正常 |

这样，你的公网地址（域名）就和你写的网页连在一起了；别人访问你的域名就能看到 Pomor 官网，并可下载安装包（若已按第四步配置）。
